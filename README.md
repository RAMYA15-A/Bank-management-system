# Bank-management-system
Bank Management System | C++ | Object-Oriented Programming
A console-based Bank Management System developed using C++ to simulate core banking operations in a structured and efficient way. This project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts and file handling for data storage.
The system allows users to create and manage bank accounts, perform basic banking operations such as deposits, withdrawals, and balance inquiries. It is designed with a simple menu-driven interface for easy interaction and better user experience.
This project helps in understanding how real-world banking systems can be modeled using programming concepts like classes, objects, encapsulation, and data persistence.
